Machine Learning Terrain Authoring

A fully functioning terrain creation tool based on a deep neural network trained with automatically processed real terrain data.

Project developed as part of Research Methods module for Teesside University.

2019

This project was an exploration into the uses of machine learning for content generation for videogames, namely in the case of terrain generation. The intent was to be able to generate detailed terrains instantly, based on only a simple sketch.

A Conditional Generative Adversarial Network model was created and trained from automatically processed real world terrain data, gathered with Houdini. A user interface tool was developed to sketch lines defining the intended terrain, run the neural network and showcase the results in real time. The pipeline can generate any number of realistic virtual terrains using the learned features from the training data, without requiring a manual definition of any procedural rules or parameter properties.

Complete Pipeline

A custom tool was developed to allow users the ability to sketch peak and valley lines that define their artistic intention and the machine learning algorithm automatically generates the resulting terrain in real time. The results are automatically exported on each change, so this could be integrated with other tools or pipelines.

A custom 3D previewer developed in OpenGL, allows rapid iteration and visualization of the result.

A simple erosion simulation is conducted and resulting terrains are then ready to be imported to any game engine and used as the basis for a 3D Landscape.

The machine learning system consists of 2 competing Networks, a generator that continuously synthesizes new terrains, and a discriminator that compares the generated ones with the real examples. The two networks compete with each other learning how to produce terrains that better match the real examples.

Machine Learing Network Outline

A system for gathering and processing the data was built in Houdini with the PDG toolset:

  • Read intended coordinates
  • Download Files from USGS servers
  • Decompress files
  • Split height information into smaller more manageable sections
  • Process each tile to compute the Peak and Valley Lines
  • Map result to matching images and save

More information on the machine learning system, the Houdini pipeline or the custom authoring tool can be found in the full dissertation bellow. The code is also available on GitHub.
