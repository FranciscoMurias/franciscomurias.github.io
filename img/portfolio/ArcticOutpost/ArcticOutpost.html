Artic Outpost (UE4)

Artic Outpost Unreal Engine 4 Scene following the theme "Lost Worlds". Developed for the 3D Games Art Degree at Teesside University.

December 2019

The unreal scene would feature a sci fi inspired small research station/communications outpost in the middle of the arctic tundra, with a massive blizzard as setting.

The outpost structure is a prefabricated modular building deployed by scientists or explorers to monitor a certain area for research. Featuring survivalist equipment, generators, radar dishes, comms towers and a heavy emphasis on functionality as part of the structure itself.

Main rule for the design: If it doesn’t have an identifiable purpose it should not be there.

After all inspiration and reference was gathered, a sketch of the proposed asset was drawn, that inspired the modelling of the final mesh. In addition to the main base, the idea of making a radar tower followed very close to the image bellow.

MODELING

DECALS

A workflow based on overlaying the base mesh with polygon cards mapped to parallax occluded decals was used to save memory while adding complex detailing to most areas of the base. This accomplished a 3D look while only using texture memory.

The decal pipeline allowed for full control over the decals color, roughness and metalness parameters separate from the base materials, or blended with an opacity parameter to overlay the underlying materials. The tertiary details of the model were constructed almost exclusively with this technique, which allowed for detail textures to remain crisp and detailed in engine, while still retaining a very optimized model when compared to the regular high to low bake workflow.

To test if the decal parallax effect would still look realistic when animated, I created a rotating fan completely using parallax materials. This was done using a rotator and the same parallax process used for the other decals to create a fan. The resulting effect looked just as good so it proved this effect can have even more applications.

I would also develop a vertex paint controlled show shader for creating snow scuffing around the base.. This proved more effective than a height based solution and allowed for finer tweaks to the more complex areas of the base but I still want to refine this effect better in the future.

FABRIC STRAPS WAVING

For the straps, I created two different UV sets for the mesh. UV0 containing a full UV of each separate quad, and UV1 combining all into a tileable texture created in photoshop from photo reference of real cloth materials. This retained an efficient texture budget, while avoiding transparency maps, making rendering even more performant.

To simulate the wind waving on the mesh, I manipulated the world position offset using the world position map from UV0 using a gradient noise map. This made it seem like wind was consistently panning through the straps. I could control the waving speed and strength in real time using exposed parameters.

BLIZZARD AND SMOKE VFX

For the vent smoke, and the blizzard I simply used the particles that come bundled with unreal in the examples of the particles demo, and tweaked some values in the particle editor. This helped add a different flair to the overall scene and make a more believable snowy blizzard environment when combined with exponential height fog.

ICICLES

I worked on a Procedural Icicles Material with 3 Blend Layers (Object, Snow, Ice) created to dynamically paint snow and icicles on any object in the scene based on vertex color.

It detects the vertex normal direction to apply a proportional tessellated height displacement creating icicles on the bottom facing areas, and snow on the top facing areas.

Light and color can be individually controlled for each blend layer, and icicles can shine through with subsurface scattering, or the same effect can be approximated with an optional emissive color exponent if desired for extra performance optimization.

AURORA

To create the effect of the aurora borealis I used an emissive material with custom color and opacity maps to simulate the rippling folds in the sky, applied to a curved mesh plane. I order to simulate movement without real animation, I used unreal’s “SimpleGrassWind” Shader to directly alter the world position offset. The included parameters proved enough to simulate a believable effect.

CONCLUSION

This project allowed for the exploration of dozens of different techniques. I will be incorporating most of them in my hard surface workflows from now on as these were really fun to utilize. The end result was interesting and very close to what I aimed for.
