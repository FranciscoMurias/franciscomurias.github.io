Runic Glyph Generator

March 2020

Python Experiment in creating an interface tool to generate user defined runic symbols.

The user can define the connection points, and procedurally generate connections to define the symbols. These can be exported as SVG shapes and manipulated further in any vector software.

The goal was to learn how to use python to make user focused authoring tools and to create a pipeline to generate any number of fictional alphabets for use in 3D projects.
