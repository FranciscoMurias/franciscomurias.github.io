Procedural Building Generator

A Houdini system for procedurally generating building structures from modular components

Project developed as part of 3D Games Art Masters Degree

2020

A building generator prototype created using Houdini. The system receives a collection of modular building parts as input and assembles the assets from a defined set of parameters to quickly generate an infinite amount of building variants.

The generator can function independently of the style of the modular assets, but was tested with a Medieval and a Mayan styled modular collection.

The generator creates random grid structures and determines placement positions, attachment points and and normal directions for the modular components. This data can also be exported independently so that the assets can be built entirely inside the game engine with minimal resource usage.
