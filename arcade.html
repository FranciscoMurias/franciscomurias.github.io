SOBRE EVENTOS GAME JAM CONTACTOS

INFO

O Porto Graphics juntou-se ao evento Future Places deste ano para te trazer a Arcade Room, uma mostra dos jogos feitos nas nossas game jams anteriores, assim como dos jogos feitos pelos estudantes de Design, Multimédia e Engenharia Informática da UP em âmbito curricular! O evento decorrerá no Maus Hábitos no dia 20 de Outubro, e será completamente livre e aberto ao público. Espera jogos, música, bebidas e muita diversão!

Participaste em alguma das nossas game jams ao longo dos anos? Andas a desenvolver um novo projeto teu? Traz o teu jogo! Só tens de preencher o formulário para teres um lugar reservado para mostrar os teus projetos!

Atenção a todos os sound designers/músicos! Falem connosco (através de facebook ou email) para trazerem as vossas game soundtracks para o evento! (As long as they are sick beats!)

Mais informações sobre o Future Places aqui.

Nota: Se tencionas mostrar o teu jogo/projecto, assegura-te que trazes todo o equipamento/computador/consola necessários para o mostrar!
